# 🎤 AI Interview Coach

> 음성 기반으로 실제 면접 흐름을 구현한 AI 면접 시뮬레이터

---

## 🚀 핵심 기능

* 🎯 **면접 질문 생성 (LLM)**
  지원 직무 기반으로 2차 임원 수준의 질문 자동 생성

* 🎤 **음성 답변 입력 (STT)**
  사용자의 답변을 음성으로 받아 텍스트로 변환

* 📊 **AI 피드백 제공**
  논리 구조, 데이터 기반 사고, 커뮤니케이션 관점에서 구조화된 피드백 생성

* 🔁 **실전 면접 흐름 구현**
  질문 → 답변 → 피드백 → 다음 질문으로 이어지는 인터랙션 설계

---

## 🧠 기술 스택

* OpenAI GPT (질문 생성 / 피드백)
* Whisper (음성 → 텍스트 변환)
* Streamlit (UI)

---
